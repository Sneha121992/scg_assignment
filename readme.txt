Steps To Install Wordpress
1. Create one database 
2. Import .sql file in database
3. Keep all files in one folder e.g scgassignment
4. Edit wp-config file and change all database credentials
   (db name, user, password, host )   
5. Go to database and edit wp-options tabel and change home_url and site_url values
